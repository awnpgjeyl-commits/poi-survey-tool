### 新的交互流程

1. 用户点击「POI 分类查询」按钮
2. 弹出模态框，显示一级分类下拉菜单
3. 选择一级分类后，显示对应的二级分类复选框列表
4. 用户勾选需要的二级分类（可多选）
5. 可在搜索框输入关键词筛选二级分类
6. 点击「确定」，分类名称自动填入搜索框
7. 点击「开始搜索」，调用百度地图 API 进行搜索
