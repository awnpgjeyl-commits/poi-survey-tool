# 部署和迁移指南

## 一、快速开始

### 1.1 本地开发

#### 前置要求
- 无需安装 Node.js、npm 或任何构建工具
- 仅需一个现代浏览器（Chrome、Firefox、Safari 等）
- 一个文本编辑器（VS Code、Sublime Text 等）

#### 步骤

1. **克隆或下载项目**:
   ```bash
   git clone https://github.com/yourusername/poi-survey-tool.git
   cd poi-survey-tool
   ```

2. **配置 API Key**:
   - 打开 `js/config.js`
   - 将 `YOUR_AMAP_KEY_HERE` 替换为你的高德地图 API Key
   - 保存文件

3. **启动本地服务器**:
   ```bash
   # 使用 Python 3
   python3 -m http.server 8000
   
   # 或使用 Python 2
   python -m SimpleHTTPServer 8000
   ```

4. **打开浏览器**:
   - 访问 `http://localhost:8000`
   - 应用应该正常加载

### 1.2 获取高德地图 API Key

1. 访问 [高德地图开放平台](https://lbs.amap.com/)
2. 注册账号（如未注册）
3. 创建应用：
   - 进入"控制台" → "应用管理" → "我的应用"
   - 点击"创建新应用"
   - 填写应用名称和应用类型（选择"Web 应用"）
   - 创建成功后获得 API Key
4. 配置 Key 安全设置：
   - 在应用详情页面，设置"服务"为"Web 服务 API"
   - 配置 IP 白名单（可选，用于生产环境）

## 二、公开发布

### 2.1 GitHub Pages（推荐）

**优点**: 免费、自动部署、支持自定义域名

**步骤**:

1. **创建 GitHub 仓库**:
   - 在 GitHub 上创建新仓库 `poi-survey-tool`
   - 选择"Public"（公开）
   - 初始化 README.md

2. **上传项目文件**:
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

3. **启用 GitHub Pages**:
   - 进入仓库设置 → "Pages"
   - 选择"Source"为"Deploy from a branch"
   - 选择分支为"main"，目录为"/ (root)"
   - 保存

4. **访问网站**:
   - 等待几分钟后，访问 `https://yourusername.github.io/poi-survey-tool`

### 2.2 Netlify

**优点**: 部署简单、支持自动构建、提供 HTTPS

**步骤**:

1. **连接 GitHub**:
   - 访问 [Netlify](https://www.netlify.com/)
   - 点击"Sign up"并选择"GitHub"登录
   - 授权 Netlify 访问你的 GitHub 账户

2. **导入项目**:
   - 点击"New site from Git"
   - 选择 GitHub 仓库 `poi-survey-tool`
   - 构建设置保持默认（无需填写构建命令）
   - 点击"Deploy site"

3. **配置环境变量**:
   - 进入"Site settings" → "Build & deploy" → "Environment"
   - 添加环境变量 `AMAP_KEY` = 你的 API Key
   - 重新部署

4. **访问网站**:
   - Netlify 会自动分配一个子域名（如 `poi-survey-tool.netlify.app`）
   - 或配置自定义域名

### 2.3 Vercel

**优点**: 部署快速、性能优异、支持自定义域名

**步骤**:

1. **连接 GitHub**:
   - 访问 [Vercel](https://vercel.com/)
   - 点击"Sign Up"并选择"GitHub"登录
   - 授权 Vercel 访问你的 GitHub 账户

2. **导入项目**:
   - 点击"New Project"
   - 选择 GitHub 仓库 `poi-survey-tool`
   - 框架选择"Other"（其他）
   - 点击"Deploy"

3. **配置环境变量**:
   - 进入项目设置 → "Environment Variables"
   - 添加 `AMAP_KEY` = 你的 API Key
   - 重新部署

4. **访问网站**:
   - Vercel 会自动分配一个子域名（如 `poi-survey-tool.vercel.app`）
   - 或配置自定义域名

### 2.4 自定义域名配置

以 GitHub Pages 为例，配置自定义域名：

1. **购买域名**:
   - 在域名注册商（如 GoDaddy、阿里云等）购买域名

2. **配置 DNS**:
   - 登录域名注册商的控制面板
   - 添加 CNAME 记录：
     - 主机记录: `www`
     - 记录值: `yourusername.github.io`
   - 或添加 A 记录指向 GitHub Pages IP

3. **在 GitHub 中配置**:
   - 进入仓库设置 → "Pages"
   - 在"Custom domain"中输入你的域名（如 `www.example.com`）
   - 保存并等待 DNS 生效

```javascript
import { useEffect, useRef, useState } from 'react';

export const useMapManager = (containerId, config) => {
  const mapRef = useRef(null);
  const [map, setMap] = useState(null);

  useEffect(() => {
    if (!window.AMap) {
      // 加载高德地图 JS API
      const script = document.createElement('script');
      script.src = `https://webapi.amap.com/maps?v=2.0&key=${config.AMAP_KEY}`;
      script.onload = () => {
        const newMap = new window.AMap.Map(containerId, {
          zoom: config.MAP_ZOOM,
          center: [116.481488, 39.990464]
        });
        setMap(newMap);
        mapRef.current = newMap;
      };
      document.body.appendChild(script);
    }
  }, [containerId, config.AMAP_KEY, config.MAP_ZOOM]);

  return {
    map,
    setCenter: (location) => map?.setCenter(location),
    drawCircle: (center, radius) => {
      const circle = new window.AMap.Circle({
        center,
        radius,
        fillColor: '#0066ff',
        fillOpacity: 0.1,
        strokeColor: '#0066ff',
        strokeWeight: 2
      });
      circle.setMap(map);
      return circle;
    },
    addMarker: (poi) => {
      const marker = new window.AMap.Marker({
        position: [poi.location.lng, poi.location.lat],
        title: poi.name
      });
      marker.setMap(map);
      return marker;
    },
    clearMarkers: () => {
      // 清空标记
    }
  };
};
```

**src/hooks/useSearchManager.js**:
```javascript
import { useState } from 'react';
import axios from 'axios';
import { CONFIG } from '../config';

export const useSearchManager = () => {
  const [results, setResults] = useState([]);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  const search = async (center, radius, poiTypes) => {
    setLoading(true);
    setError(null);
    try {
      const response = await axios.get(CONFIG.AMAP_API_ENDPOINTS.AROUND_SEARCH, {
        params: {
          location: `${center.lng},${center.lat}`,
          radius: radius * 1000,
          types: poiTypes.join('|'),
          pagesize: 20,
          pagenum: 1,
          extensions: 'all',
          key: CONFIG.AMAP_KEY
        }
      });

      if (response.data.status === '1') {
        const pois = response.data.pois || [];
        setResults(pois);
      } else {
        setError('搜索失败，请重试');
      }
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  };

  return { results, loading, error, search };
};
```

```javascript
import React, { useState } from 'react';
import { CONFIG } from '../config';

export const SidebarPanel = ({ onSearch }) => {
  const [address, setAddress] = useState('');
  const [radius, setRadius] = useState(5);
  const [selectedCategories, setSelectedCategories] = useState([]);
  const [customPOI, setCustomPOI] = useState('');

  const handleSearch = () => {
    onSearch({ address, radius, selectedCategories, customPOI });
  };

  const toggleCategory = (category) => {
    setSelectedCategories(prev =>
      prev.includes(category)
        ? prev.filter(c => c !== category)
        : [...prev, category]
    );
  };

  return (
    <div className="sidebar">
      <div className="search-box">
        <input
          type="text"
          placeholder="输入地址"
          value={address}
          onChange={(e) => setAddress(e.target.value)}
        />
      </div>

      <div className="radius-control">
        <label>搜索半径: {radius} km</label>
        <input
          type="range"
          min="1"
          max="10"
          value={radius}
          onChange={(e) => setRadius(Number(e.target.value))}
        />
      </div>

      <div className="category-selector">
        <h3>业态分类</h3>
        {Object.keys(CONFIG.POI_CATEGORIES).map(category => (
          <label key={category}>
            <input
              type="checkbox"
              checked={selectedCategories.includes(category)}
              onChange={() => toggleCategory(category)}
            />
            {category}
          </label>
        ))}
      </div>

      <div className="custom-poi">
        <h3>其他 POI</h3>
        <input
          type="text"
          placeholder="输入 POI 编码，如: 120201,200100"
          value={customPOI}
          onChange={(e) => setCustomPOI(e.target.value)}
        />
      </div>

      <button onClick={handleSearch}>搜索</button>

  );
};
```

**src/App.jsx**:
```javascript
import React, { useState } from 'react';
import { MapContainer } from './components/MapContainer';
import { SidebarPanel } from './components/SidebarPanel';
import { ResultsTable } from './components/ResultsTable';
import { useSearchManager } from './hooks/useSearchManager';
import './App.css';

function App() {
  const [center, setCenter] = useState({ lng: 116.481488, lat: 39.990464 });
  const [radius, setRadius] = useState(5);
  const { results, search } = useSearchManager();

  const handleSearch = async ({ address, radius, selectedCategories, customPOI }) => {
    // 地址转坐标
    // 调用搜索
    setRadius(radius);
    const poiTypes = [
      ...selectedCategories.flatMap(cat => CONFIG.POI_CATEGORIES[cat] || []),
      ...customPOI.split(',').filter(code => code.trim())
    ];
    await search(center, radius, poiTypes);
  };

  return (
    <div className="app">
      <div className="container">
        <SidebarPanel onSearch={handleSearch} />
        <div className="main-content">
          <MapContainer center={center} radius={radius} />
          <ResultsTable results={results} />
        </div>
      </div>
    </div>
  );
}

export default App;
```

### Q1: 如何更新 POI 分类映射表？

### Q2: 如何处理 API 限流？

实现请求队列和缓存机制：

```javascript
class RequestQueue {
  constructor(maxConcurrent = 5) {
    this.maxConcurrent = maxConcurrent;
    this.queue = [];
    this.running = 0;
  }

  async add(fn) {
    return new Promise((resolve, reject) => {
      this.queue.push({ fn, resolve, reject });
      this.process();
    });
  }

  async process() {
    if (this.running >= this.maxConcurrent || this.queue.length === 0) {
      return;
    }

    this.running++;
    const { fn, resolve, reject } = this.queue.shift();

    try {
      const result = await fn();
      resolve(result);
    } catch (error) {
      reject(error);
    } finally {
      this.running--;
      this.process();
    }
  }
}
```

### Q3: 如何支持离线模式？

使用 Service Worker 缓存数据：

```javascript
// 在 app.js 中注册 Service Worker
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js');
}
```

### Q4: 如何提高搜索性能？

- 实现地理编码缓存
- 使用虚拟滚动显示大量结果
- 实现增量搜索（用户输入时实时搜索）
